# 🔐 Cybersecurity Professional Portfolio

## Welcome to My Digital Security Showcase

This is a professional portfolio website showcasing my expertise in cybersecurity, penetration testing, vulnerability assessment, and security architecture.

### ⭐ Features

- **Modern, Responsive Design** - Looks great on all devices
- **Dark Theme** - Easy on the eyes with a professional cybersecurity aesthetic
- **Smooth Animations** - Beautiful transitions and effects
- **Mobile Optimized** - Full functionality on mobile devices
- **SEO Friendly** - Built with best practices
- **Contact Form** - Easy way to get in touch

### 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/JonathanXiaomi999/portfolio.git
```

2. Open `index.html` in your browser

3. Or enable GitHub Pages:
   - Go to repository Settings
   - Navigate to Pages
   - Select `main` branch as source
   - Your site will be live at: `https://JonathanXiaomi999.github.io/portfolio/`

### 📝 Customization

To customize this portfolio:

1. **Edit Contact Information:**
   - Update email in contact section
   - Replace phone number
   - Update location

2. **Update Projects:**
   - Modify project descriptions
   - Update GitHub repository links
   - Add/remove project cards

3. **Customize Colors:**
   - Edit CSS variables in `styles.css`
   - Change primary color (#FF0000)
   - Adjust accent color (#00d4ff)

4. **Add Social Links:**
   - Update LinkedIn profile URL
   - Add/modify social media handles
   - Include additional platforms

5. **Update Certifications:**
   - Add/remove certification cards
   - Update certification names and years
